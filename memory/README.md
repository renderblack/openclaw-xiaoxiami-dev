# 内存管理系统

## 目录结构说明

### 📁 projects/ - 项目相关记忆
- 按项目名称创建子目录
- 存储项目特定的配置、进展、问题记录
- 例如：`projects/openclaw/`、`projects/webapp/`

### 📁 templates/ - 模板文件
- 常用模板：会议记录、项目计划、问题报告等
- 可复用的记忆结构

### 📁 archive/ - 归档记忆
- 已完成或不活跃项目的记忆
- 按年份-月份归档：`archive/2026-03/`

### 📁 temp/ - 临时记忆
- 短期记忆，需要定期清理
- 会话缓存、临时笔记等

## 使用规范
1. 每日记忆仍保存在根目录 `YYYY-MM-DD.md`
2. 项目相关记忆按需存入对应项目目录
3. 每月初检查并归档上月记忆
4. 临时记忆每周清理一次
